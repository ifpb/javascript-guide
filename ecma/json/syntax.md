## Creating data sets

---

| NAME    | EMAIL             |
| ------- | ----------------- |
| fulano  | fulano@gmail.com  |
| sicrano | sicrano@gmail.com |

### Comma Separated Values (CSV)

```csv
name, email
fulano, fulano@gmail.com
sicrano, sicrano@gmail.com
```

### eXtensible Markup Language (XML)

```xml
<students>
  <student>
    <name>Fulano</name>
    <email>fulano@gmail.com</email>
  </student>
  <student>
    <name>Sicrano</name>
    <email>sicrano@gmail.com</email>
  </student>
</students>
```

### YAML Ain't Markup Language (YAML)

Reference: [doc](http://yaml.org/)

```yaml
- name: 'fulano'
  email: 'fulano@gmail.com'
- name: 'sicrano'
  email: 'sicrano@gmail.com'
```

### Javascript Object Notation (JSON)

![Object](https://www.json.org/img/object.png)

![Value](https://www.json.org/img/value.png)

Reference: [JSON ORG](http://json.org)

#### Quoted (.json, .js)

```js
[
  {
    "name": "fulano",
    "email": "fulano@gmail.com",
  },
  {
    "name": "sicrano",
    "email": "sicrano@gmail.com",
  }
]
```

#### Unquoted (.js)

```js
const students = [
  {
    name: 'fulano',
    email: 'fulano@gmail.com',
  },
  {
    name: 'sicrano',
    email: 'sicrano@gmail.com',
  },
];
```

#### Indexed & Unquoted (.js)

```js
const students = {
  20181234: {
    name: 'fulano',
    email: 'fulano@gmail.com'
  },
  20181235: {
    name: 'sicrano',
    email: 'sicrano@gmail.com'
  }
}
```

- [json.org](http://json.org/)
